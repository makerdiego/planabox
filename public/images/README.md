# Imágenes necesarias

Coloca las siguientes imágenes en este directorio:

1. **hero-planabox.png** (1200x800px recomendado)
   - Imagen principal del hero
   - Se muestra en la sección principal de la landing page

2. **sizes-illustration.png** (800x400px recomendado)
   - Ilustración de los diferentes tamaños de trasteros
   - Se muestra en la sección "¿Qué cabe en cada tamaño?"

3. **security-icon.png** (400x400px recomendado)
   - Icono o ilustración relacionada con seguridad
   - Se muestra en la sección "Seguridad y acceso"

4. **og-planabox.png** (1200x630px recomendado)
   - Imagen para Open Graph (redes sociales)
   - Se muestra cuando se comparte el enlace en WhatsApp, Facebook, Twitter, etc.

## Formato recomendado
- PNG o JPG
- Optimizadas para web (comprime las imágenes antes de subirlas)
- Usa herramientas como TinyPNG o ImageOptim para reducir el tamaño de archivo
